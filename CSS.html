What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls the layout, design, and visual aspects of web pages, such as colors, fonts, spacing, and positioning. CSS was created by Håkon Wium Lie and Bert Bos in 1994, and it has since become a crucial part of web development, allowing for separation of content (HTML) and design (CSS) in web documents.

History and Evolution

  • CSS1 (1996): The first version, focused on basic styling features such as fonts, colors, and alignment.

  • CSS2 (1998): Introduced advanced features like positioning, media types, and more precise control over page layout.

  • CSS3 (1999-Present): Split into multiple modules, adding support for animations, transitions, gradients, and responsive design.

    • What is CSS used for?

    Styling web pages: CSS defines the appearance of elements on a webpage, such as text color, background images, borders, and layout.
    Layout management: CSS is used to arrange elements on a page using tools like grids, flexbox, and positioning techniques.
    Responsiveness: CSS allows websites to adapt their layout to different screen sizes, ensuring a good user experience on desktops, tablets, and mobile devices.
    Enhancing user interface: CSS is used to create hover effects, transitions, and animations that improve the interaction between the user and the webpage.
    Customizing web elements: With CSS, developers can control the look of forms, buttons, navigation menus, and other interface components.

    CSS Tags Explained

    1. color - Text Color

    The color property is used to set the text color of an element.

    p { color: red; }

    This paragraph has red text color.

    2. background-color - Background Color

    The background-color property is used to set the background color of an element.

    div { background-color: lightblue; }

    This div has a light blue background.

    3. font-size - Font Size

    The font-size property specifies the size of the font for the text within an element.

    h1 { font-size: 36px; }

    This header has a font size of 36px.

    4. margin - External Spacing

    The margin property creates space outside the border of an element.

    div { margin: 20px; }

    This div has a margin of 20px around it.

    5. padding - Internal Spacing

    The padding property adds space inside the element, between its content and the border.

    div { padding: 15px; }

    This div has 15px padding inside it.

    6. border - Element Border

    The border property sets the border style, width, and color of an element.

    div { border: 2px solid black; }

    This div has a 2px solid black border.

    7. display - Display Type

    The display property defines the display type of an element. Common values include block, inline, and flex.

    div { display: flex; }

    This div uses the flex display property, making its child elements flexible.
