What is SVG?

SVG (Scalable Vector Graphics) is an XML-based file format used to define vector graphics for the web. It allows for the creation of interactive and scalable images such as logos, icons, illustrations, and diagrams. Unlike raster images (like JPEG or PNG), SVG images can be resized without losing quality, making them ideal for responsive web design. SVG was developed by the W3C (World Wide Web Consortium) and became a standard in 2001.

History and Evolution

  • SVG 1.0 (2001): The first version, focused on defining vector graphics and basic shapes like lines, circles, and paths.

  • SVG 1.1 (2003): Expanded functionality, allowing for more advanced shapes, filters, and animations.

  • SVG 2.0 (Upcoming): The next version is expected to enhance support for features like better styling, accessibility, and improved interactivity with JavaScript.

    • What is SVG used for?

    Creating scalable graphics: SVG images can be resized without losing quality, making them ideal for responsive web design.
    Web icons and illustrations: SVG is widely used for creating clean and simple icons, logos, and illustrations that remain sharp on all screen sizes.
    Animations and interactivity: SVG supports animation and interactivity via CSS and JavaScript, allowing for dynamic visual elements on a page.
    Data visualization: SVG is often used to create charts, graphs, and other data visualizations that scale smoothly across devices.
    Accessibility: SVG elements are accessible to screen readers, and their text can be styled and interacted with using web technologies.

    1. <svg>

    The main container for SVG graphics, which defines the space where the content will be drawn.

    <svg width="200" height="200" style="border: 1px solid black;">
                    <circle cx="100" cy="100" r="50" fill="blue" />
                </svg>

    2. <circle>

    Used to draw circles. It requires the attributes cx, cy (center coordinates), and r (radius).

    <svg width="200" height="200">
                    <circle cx="100" cy="100" r="40" fill="red" />
                </svg>

    3. <rect>

    Draws a rectangle. Uses the attributes x, y (starting position), and width, height (dimensions).

    <svg width="200" height="200">
                    <rect x="50" y="50" width="100" height="100" fill="green" />
                </svg>

    4. <line>

    Draws a straight line between two points. It requires the attributes x1, y1 (start), and x2, y2 (end).

    <svg width="200" height="200">
                    <line x1="10" y1="10" x2="150" y2="150" stroke="purple" stroke-width="2" />
                </svg>

    5. <polygon>

    Draws a polygon using a list of connected points.

    <svg width="200" height="200">
                    <polygon points="50,150 100,50 150,150" fill="orange" />
                </svg>

    6. <path>

    Allows drawing complex shapes with a command sequence in the d attribute.

    <svg width="200" height="200">
                    <path d="M10 80 C 40 10, 65 10, 95 80 S 150 150, 180 80" stroke="black" fill="transparent" />
                </svg>

    7. <text>

    Draws text within the SVG. Uses x and y to position the text.

    <svg width="200" height="200">
                    <text x="50" y="100" font-family="Verdana" font-size="25" fill="blue">Hello SVG!</text>
                </svg>
