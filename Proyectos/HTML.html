    HTML Tags Explained

    1. <h1> - Main Header

    The <h1> tag is used to create a header. You can choose header 1,2 or 3 to change the size

    <h1>This is a main header</h1>

    2. <p> - Paragraph

    The <p> tag is used to create a paragraph

    <p>This is a paragraph of text in HTML.</p>

    3. <a> - Anchor (Link)

    The <a> tag is used to create links to other pages or resources. It is used with the href attribute to specify the destination URL.

    <a href="https://www.example.com">Visit our website</a>

    4. <img> - Image

    The <img> tag is used to embed images in a webpage. It requires the src attribute to specify the image source and the alt attribute for an alternative description of the image.

    <img src="image.jpg" alt="A descriptive image" />

    5. <ul> and <li> - Unordered List

    The <ul> tag creates an unordered list, and each list item is defined inside a <li> tag.

    <ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

    6. <ol> and <li> - Ordered List

    The <ol> tag creates an ordered list, with list items automatically numbered. Each item is defined inside a <li> tag.

    <ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

    7. <form> - Form

    The <form> tag is used to create forms that allow users to submit data to a server. Forms contain elements like <input>, <label>, and <button>.

    <form action="/submit_form" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" />
  <button type="submit">Submit</button>
</form>

    8. <div> - Division/Container

    The <div> tag is a generic container used to group other HTML elements. It is commonly used for styling and structuring a webpage.

    <div>
  <h2>Header inside a div</h2>
  <p>This paragraph is inside a div container.</p>
</div>

    9. <table>, <tr>, <th>, <td> - Table

    The <table>, <tr>, <th>, and <td> tags are used to create tables in HTML. <tr> defines a row, <th> defines a header cell, and <td> defines a data cell.

    <table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Juan</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Ana</td>
    <td>30</td>
  </tr>
</table>

    10. <span> - Inline Element

    The <span> tag is used to group inline content, often for applying styles to part of a text without affecting the overall structure of the document.

    <p>Juan's favorite color is <span style="color:red">red</span>.</p>
